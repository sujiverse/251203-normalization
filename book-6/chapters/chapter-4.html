변수부터 회귀분석까지

    1부: 연구와 데이터의 기초

    [4장] 변수의 역할별 분류

    이 장에서는 연구에서 변수가 맡는 역할에 따른 분류를 배웁니다. 독립변수, 종속변수부터 매개변수, 조절변수까지 완전히 이해합니다.

    1. 독립변수 (Independent Variable)

    독립변수는 다른 변수에 영향을 미치는 원인 또는 예측자로 가정되는 변수입니다.

    정의와 특성

    • 다른 이름: 예측변수(Predictor), 설명변수(Explanatory variable), 처리변수(Treatment)
    • 위치: 인과 모형에서 화살표의 시작점
    • 표기: 보통 X로 표기 (X → Y)

    실험 연구에서의 독립변수

    연구자가 의도적으로 조작(manipulation)합니다.

    • 참가자를 조건에 무선 할당
    • 처치의 유무나 종류를 결정
    • 예: 약물 투여 여부, 교육 프로그램 유형
    participant_idtreatmentoutcome_score
    1experimental85
    2control72
    3experimental88

    비실험 연구에서의 독립변수

    연구자가 조작하지 않고 있는 그대로 관찰합니다.

    • 참가자의 기존 특성을 측정
    • 인과관계 추론에 제한이 있음
    • 예: 성별, 연령, 흡연 여부

    주의: 비실험 연구에서 "독립변수"라는 용어 대신 "예측변수"를 사용하는 것이 더 정확합니다.

    2. 종속변수 (Dependent Variable)

    종속변수는 독립변수의 영향을 받는 결과로 가정되는 변수입니다.

    정의와 특성

    • 다른 이름: 결과변수(Outcome), 반응변수(Response), 준거변수(Criterion)
    • 위치: 인과 모형에서 화살표의 끝점
    • 표기: 보통 Y로 표기 (X → Y)
    • 핵심: 연구자가 설명하거나 예측하고자 하는 변수

    종속변수 선정 기준

    • 타당성: 연구 질문에서 알고자 하는 것을 반영
    • 신뢰성: 일관되게 측정 가능
    • 민감성: 독립변수의 변화를 감지할 수 있음
    • 실용성: 측정이 현실적으로 가능

    예시: 물리치료 효과 연구

    • 독립변수: 치료 유형 (실험군: 신규 치료 / 대조군: 기존 치료)
    • 종속변수:
      • 통증 강도 (VAS 0-100)
      • 기능 장애 (ODI 0-100%)
      • 삶의 질 (SF-36 점수)

    3. 통제변수 (Control Variable)

    통제변수는 종속변수에 영향을 미칠 수 있어서 그 영향을 통제하는 변수입니다.

    왜 통제하는가?

    독립변수와 종속변수의 관계를 더 정확하게 파악하기 위해서입니다.

    • 제3의 변수가 결과에 영향을 미치면 독립변수의 진정한 효과를 알 수 없음
    • 통제를 통해 "다른 조건이 같을 때" 독립변수의 효과를 분리

    통제 방법

    방법설명예시
    실험적 통제 무선 할당으로 집단 간 동등성 확보 치료군/대조군에 무작위 배정
    제한/배제 특정 조건의 참가자만 포함 여성만 연구에 포함
    짝짓기 집단 간 통제변수를 동일하게 나이, 성별을 매칭
    통계적 통제 분석 시 공변량으로 포함 ANCOVA, 회귀분석에서 통제

    DB 설계: 통제변수 저장

    CREATE TABLE participants (
    id INTEGER PRIMARY KEY,
    condition TEXT,        -- 독립변수: 실험 조건

    -- 통제변수들 (공변량)
    age INTEGER,
    gender TEXT,
    bmi REAL,
    baseline_pain REAL,    -- 사전 통증 점수

    -- 종속변수
    post_pain REAL         -- 사후 통증 점수
);

-- 통계적 통제를 위한 쿼리 준비
-- ANCOVA나 회귀분석에서 age, gender, bmi, baseline_pain을 공변량으로 사용

    4. 매개변수 (Mediator)

    매개변수는 독립변수가 종속변수에 영향을 미치는 경로상의 중간 변수입니다.

    매개효과의 구조

    X → M → Y

    • X (독립변수): 원인
    • M (매개변수): 메커니즘, "어떻게"
    • Y (종속변수): 결과

    매개변수는 "왜" 효과가 발생하는지의 메커니즘을 설명합니다.

    예시: 운동 → 자기효능감 → 우울 감소

    • X (운동): 운동 프로그램 참여
    • M (자기효능감): 자신감 향상
    • Y (우울): 우울 증상 감소

    운동이 우울을 줄이는 이유 중 하나가 자기효능감 향상일 수 있습니다.

    완전 매개 vs 부분 매개

    유형설명경로
    완전 매개
    (Full Mediation)    		 X→Y 직접 효과가 없어짐
    모든 효과가 M을 통해 전달    		 X → M → Y
    (X → Y = 0)
    부분 매개
    (Partial Mediation)    		 X→Y 직접 효과도 존재
    일부 효과가 M을 통해 전달    		 X → M → Y
    X → Y (감소하지만 존재)

    매개효과 검증 조건 (Baron & Kenny, 1986)

    1. X → Y 관계가 유의해야 함 (총효과)
    2. X → M 관계가 유의해야 함
    3. M → Y 관계가 유의해야 함 (X 통제 상태에서)
    4. M을 투입하면 X → Y 관계가 감소 또는 사라짐

    참고: 현대적 방법은 부트스트래핑(Bootstrapping)으로 간접효과의 유의성을 직접 검정합니다.

    5. 조절변수 (Moderator)

    조절변수는 독립변수와 종속변수 간 관계의 강도나 방향을 변화시키는 변수입니다.

    조절효과의 구조

    X → Y 관계가 W의 값에 따라 달라집니다.

    • X (독립변수): 원인
    • W (조절변수): "언제", "누구에게"
    • Y (종속변수): 결과
    • X × W (상호작용): 조절효과를 나타내는 항

    조절변수는 "언제" 또는 "누구에게" 효과가 다른지를 설명합니다.

    예시: 치료 효과의 성별 차이

    • X (치료): 신규 치료 vs 기존 치료
    • W (성별): 남성 vs 여성
    • Y (통증): 통증 감소

    치료 효과가 남성에서는 크고 여성에서는 작다면, 성별이 조절변수입니다.

    조절효과 검증: 상호작용 항

    회귀분석에서 X × W 상호작용 항이 유의한지 검정합니다.

    Y = b₀ + b₁X + b₂W + b₃(X × W) + ε

-- b₃가 유의하면 조절효과가 있음

    해석: X가 Y에 미치는 효과가 W의 수준에 따라 달라짐

    매개변수 vs 조절변수

    구분매개변수 (Mediator)조절변수 (Moderator)
    질문"어떻게" 효과가 발생하나?"언제/누구에게" 효과가 다른가?
    경로X → M → Y (순차적)X × W → Y (상호작용)
    인과 구조M은 X와 Y 사이에 위치W는 X-Y 관계를 수정
    분석 방법매개분석, 부트스트래핑상호작용 분석, 조건부 효과
    예시운동 → 자기효능감 → 우울치료 × 성별 → 통증

    6. 혼입변수 (Confounding Variable)

    혼입변수(교란변수)는 독립변수와 종속변수 모두에 영향을 미쳐 허위 관계를 만드는 변수입니다.

    혼입의 구조

        C (혼입변수)
   ↙     ↘
  X   →   Y  (허위 관계)
    • C → X: 혼입변수가 독립변수에 영향
    • C → Y: 혼입변수가 종속변수에 영향
    • X → Y: 실제로는 인과관계가 없거나 약할 수 있음

    예시: 아이스크림 판매와 익사 사고

    • X (아이스크림 판매량): 여름에 증가
    • Y (익사 사고): 여름에 증가
    • C (기온/계절): 둘 다에 영향

    아이스크림 판매와 익사 사고는 상관이 있지만, 아이스크림이 익사를 유발하지 않습니다. 기온(계절)이 혼입변수입니다.

    혼입변수의 조건

    1. 독립변수(X)와 관련이 있어야 함
    2. 종속변수(Y)에 영향을 미쳐야 함
    3. 독립변수(X)의 결과가 아니어야 함 (X → C가 아님)

    혼입 통제 방법

    방법설명
    무선 할당 실험 연구에서 가장 강력한 방법. 알려진/알려지지 않은 혼입변수 모두 통제
    제한 혼입변수의 특정 수준만 포함 (예: 남성만)
    짝짓기 비교 집단 간 혼입변수를 동일하게 맞춤
    층화 혼입변수 수준별로 나누어 분석
    통계적 조정 회귀분석에서 공변량으로 포함

    통제변수 vs 혼입변수

    • 혼입변수: 통제하지 않으면 결과를 왜곡 (문제)
    • 통제변수: 분석에서 그 영향을 제거하고자 하는 변수 (해결책)

    혼입변수는 통제해야 할 대상이고, 통제변수는 통제하는 행위를 반영합니다. 혼입변수를 통제변수로 포함하여 분석합니다.

    7. 변수 역할 관계도

    연구 모형에서 각 변수의 역할을 한눈에 정리합니다.

    전체 관계도

                        혼입변수 (C)
                   ↙         ↘
  통제변수 (CV)    독립변수 (X) ──────→ 종속변수 (Y)
       ↘              ↓              ↑
        ──────→  매개변수 (M) ────→
                      ↑
                 조절변수 (W)
                (X × W 상호작용)
    • X → Y: 직접 효과
    • X → M → Y: 간접 효과 (매개)
    • X × W → Y: 조절 효과
    • C → X, C → Y: 혼입 (통제 필요)
    • CV → Y: 통제변수의 영향 (분석에서 제거)

    예시: 운동 프로그램 연구 모형

    역할변수설명
    독립변수 (X)운동 프로그램실험군: 운동 / 대조군: 미참여
    종속변수 (Y)우울 점수PHQ-9로 측정
    매개변수 (M)자기효능감운동이 자기효능감을 높여 우울 감소
    조절변수 (W)사회적 지지지지가 높으면 운동 효과 증가
    통제변수 (CV)나이, 성별, 기저 우울집단 간 차이 통제
    잠재적 혼입 (C)기저 신체 활동 수준운동 참여와 우울 모두에 영향

    DB 설계: 변수 역할 메타데이터

    CREATE TABLE study_variables (
    id INTEGER PRIMARY KEY,
    variable_name TEXT NOT NULL,
    variable_label TEXT,
    role TEXT CHECK(role IN (
        'independent', 'dependent', 'control',
        'mediator', 'moderator', 'confound'
    )),
    data_type TEXT,
    measurement_instrument TEXT
);

INSERT INTO study_variables VALUES

(4, 'social_support', '사회적 지지', 'moderator', 'continuous', 'MSPSS'),
(5, 'age', '나이', 'control', 'continuous', NULL),

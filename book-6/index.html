📘 6권: 연구 데이터 설계

변수부터 회귀분석까지

    연구자를 위한 데이터 설계 완전정복

    "내 연구 데이터, 어떻게 정리해야 할지 모르겠어요"

    이 책은 연구 설계부터 회귀분석까지, 데이터를 어떻게 구조화하고 저장해야 하는지를 체계적으로 다룹니다. 엑셀의 한계를 넘어, 재현 가능하고 분석하기 좋은 데이터 구조를 만드는 법을 배웁니다.

    이 책을 쓴 이유

    많은 연구자들이 이런 고민을 합니다:

    • 📁 "데이터 파일이 10개가 넘는데 어떤 게 최신인지 모르겠어요"
    • 📊 "SPSS에 넣으려니까 형식이 안 맞아요"
    • 🔄 "분석할 때마다 데이터를 다시 정리해야 해요"
    • ❓ "이 변수는 어떤 타입으로 저장해야 하죠?"
    • 📈 "회귀분석 돌리려면 데이터가 어떤 구조여야 하나요?"

    이 책은 이런 질문들에 대한 체계적인 답을 제공합니다.

    누구를 위한 책인가요?

    • 📌 학위 논문을 준비하는 대학원생
    • 📌 연구 프로젝트를 진행하는 연구원
    • 📌 데이터 수집/관리를 담당하는 연구보조원
    • 📌 통계 분석을 배우고 싶은 모든 분
    • 📌 엑셀의 한계를 느낀 모든 연구자

    전체 구성: 5부 20장

    1부: 연구와 데이터의 기초 (1-4장)
    장제목핵심 내용
    1장엑셀이 연구를 망치는 이유Wide vs Long, 복붙의 저주
    2장연구 설계의 기본 구조모집단, 표본, 가설
    3장변수란 무엇인가조작적 정의, 측정 오차
    4장변수의 역할별 분류독립/종속/매개/조절/통제
    2부: 변수 유형과 저장 원칙 (5-9장)
    장제목핵심 내용
    5장척도의 4가지 수준명목/서열/등간/비율
    6장범주형 변수 완전정복코드화, 더미변수
    7장연속형 변수 완전정복정규성, 변환, 표준화
    8장특수 변수 처리결측치, 이상치, 날짜
    9장복합 변수와 척도역코딩, 신뢰도
    3부: 연구 형태별 DB 설계 (10-14장)
    장제목핵심 내용
    10장횡단 연구1회 측정, 비교집단
    11장종단 연구반복측정, 패널, 코호트
    12장실험 연구피험자간/피험자내/혼합
    13장설문 연구문항 정규화, 리커트
    14장질적/혼합 연구코딩, 테마, 통합
    4부: 기술통계와 추론통계 (15-18장)
    장제목핵심 내용
    15장기술통계의 모든 것평균, 표준편차, 분포
    16장집단 비교 분석t-test, ANOVA, 비모수
    17장관계 분석상관, 교차분석
    18장회귀분석단순/다중/위계적 회귀
    5부: 실전 데이터 관리 (19-20장)
    장제목핵심 내용
    19장코드북과 메타데이터변수 정의, 값 레이블
    20장분석 도구 연동SPSS/R/Python 연동

## 챕터별 소제목 구조

### 1부: 연구와 데이터의 기초 (1-4장)

#### 1장: 엑셀이 연구를 망치는 이유
- 연구자의 흔한 고통
- Wide vs Long 형식
- 복붙의 저주: 데이터 불일치
- 재현 불가능한 분석
- 데이터베이스가 해결책인 이유

#### 2장: 연구 설계의 기본 구조
- 연구 질문에서 시작하기
- 모집단과 표본
- 가설 설정 (귀무가설, 대립가설)
- 연구 설계 → DB 구조 매핑
- 좋은 연구 설계의 조건

#### 3장: 변수란 무엇인가
- 변수의 정의
- 개념적 정의 vs 조작적 정의
- 측정과 측정 오차
- 신뢰도와 타당도 개요
- 변수명 짓기 규칙

#### 4장: 변수의 역할별 분류
- 독립변수 (Independent Variable)
- 종속변수 (Dependent Variable)
- 통제변수 (Control Variable)
- 매개변수 (Mediator)
- 조절변수 (Moderator)
- 혼입변수 (Confounding Variable)
- 역할 관계도

---

### 2부: 변수 유형과 저장 원칙 (5-9장)

#### 5장: 척도의 4가지 수준
- 명목 척도 (Nominal)
- 서열 척도 (Ordinal)
- 등간 척도 (Interval)
- 비율 척도 (Ratio)
- 척도별 가능한 연산
- 척도별 적합한 통계 분석
- DB 데이터 타입 매핑

#### 6장: 범주형 변수 완전정복
- 이분형 변수 (Binary)
- 다분형 변수 (Multinomial)
- 순서형 변수 (Ordinal)
- 코드화와 레이블 테이블
- 다중응답 변수 처리
- 더미변수 변환

#### 7장: 연속형 변수 완전정복
- 정수 vs 실수
- 정밀도와 반올림
- 단위 관리
- 정규분포와 비정규분포
- 변환 (로그, 제곱근, 역수)
- 표준화 (Z-score)

#### 8장: 특수 변수 처리
- 날짜와 시간 변수
- 결측치의 3가지 유형 (MCAR, MAR, MNAR)
- 결측치 처리 방법
- 이상치 탐지와 처리
- 텍스트 변수
- 파일 경로 저장

#### 9장: 복합 변수와 척도
- 단일 문항 vs 다문항 척도
- 총점과 평균 계산
- 역코딩 (Reverse Coding)
- 신뢰도: Cronbach's α
- 요인분석 개요
- 척도 점수 저장 전략

---

### 3부: 연구 형태별 DB 설계 (10-14장)

#### 10장: 횡단 연구 (Cross-sectional)
- 횡단 연구란?
- 1회 측정 구조
- 참가자-응답 테이블 설계
- 비교 집단 설계
- 횡단 연구의 한계

#### 11장: 종단 연구 (Longitudinal)
- 종단 연구의 종류 (패널, 코호트, 추세)
- 반복 측정 데이터
- Wide vs Long 형식 비교
- 시점(wave) 관리
- 탈락(attrition) 처리
- 종단 DB 설계 예시

#### 12장: 실험 연구 (Experimental)
- 실험 설계의 기본
- 피험자간 설계 (Between-subjects)
- 피험자내 설계 (Within-subjects)
- 혼합 설계 (Mixed design)
- 무선화와 균형화
- 조건-시행-측정 테이블 구조

#### 13장: 설문 연구 (Survey)
- 설문 문항 정규화
- 리커트 척도 저장
- 조건 분기 문항
- 매트릭스 문항
- 개방형 문항
- 설문 DB 설계 예시

#### 14장: 질적/혼합 연구 (Qualitative/Mixed)
- 질적 연구 데이터의 특성
- 인터뷰 전사 저장
- 코딩과 코드 테이블
- 테마 계층 구조
- 양적-질적 통합 전략
- 혼합 연구 DB 설계

---

### 4부: 기술통계와 추론통계 (15-18장)

#### 15장: 기술통계의 모든 것
- 중심 경향: 평균, 중앙값, 최빈값
- 산포: 범위, 분산, 표준편차, IQR
- 분포 형태: 왜도, 첨도
- 빈도와 백분율
- 기술통계 SQL 쿼리

#### 16장: 집단 비교 분석
- 분석 방법 선택 의사결정 트리
- 독립표본 t-test
- 대응표본 t-test
- 일원분산분석 (One-way ANOVA)
- 이원분산분석 (Two-way ANOVA)
- 반복측정 ANOVA
- 비모수 검정 (Mann-Whitney, Kruskal-Wallis, Wilcoxon)
- 효과크기

#### 17장: 관계 분석
- 피어슨 상관 (Pearson r)
- 스피어만 상관 (Spearman ρ)
- 편상관
- 교차분석 (χ² 검정)
- 상관 매트릭스
- 상관 vs 인과

#### 18장: 회귀분석
- 회귀분석이란?
- 단순 선형 회귀
- 다중 회귀
- 위계적 회귀
- 회귀분석의 가정
- 다중공선성 (VIF)
- 회귀 결과 해석
- 로지스틱 회귀 개요

#### 19장: 코드북과 메타데이터

- 변수명 규칙

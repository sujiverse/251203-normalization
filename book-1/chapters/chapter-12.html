데이터베이스 기초 개념

    [부록] MySQL + DBeaver 초기 세팅 가이드

    1. MySQL 설치

    Windows

    1. dev.mysql.com/downloads/installer 접속
    2. "MySQL Installer for Windows" 다운로드
    3. 설치 시 "Developer Default" 선택
    4. root 비밀번호 설정 → 꼭 기억하세요!

    Mac (Homebrew)

    brew install mysql
brew services start mysql
mysql_secure_installation

    2. DBeaver 설치

    1. dbeaver.io/download 접속
    2. 운영체제에 맞는 버전 다운로드
    3. 설치 진행

    3. DBeaver에서 MySQL 연결

    Server Host: localhost
Port:        3306
Username:    root
Password:    (설치할 때 설정한 비밀번호)

    4. 데이터베이스 생성

    CREATE DATABASE pt_clinic
DEFAULT CHARACTER SET utf8mb4;

USE pt_clinic;

    🎉 완료!

    이제 물리치료 클리닉 DB를 구축할 준비가 되었습니다!

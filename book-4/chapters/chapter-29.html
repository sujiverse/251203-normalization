    문제 해결 가이드

    서버가 안 켜져요

    1. 포트 충돌 확인
   netstat -an | findstr 5000

2. 다른 프로세스가 5000 포트 사용 중이면 종료
   또는 다른 포트로 변경

3. Python 오류 확인
   python app.py  # 에러 메시지 확인

    DB 연결 오류

    1. MySQL 서비스 실행 확인
   - 서비스 → MySQL 시작

2. 연결 정보 확인
   - .env 파일의 DB_HOST, DB_PASSWORD 확인

3. 직접 접속 테스트
   mysql -u root -p healing_hands

    화면이 느려요

    1. 데이터 양 확인
   - 너무 많은 날짜 범위를 한 번에 표시하면 느림
   - 2주 정도가 적당

2. 인덱스 확인 (3권 참고)

3. 브라우저 캐시 지우기

    예약이 안 보여요

    1. 날짜 범위 확인
   - 예약이 있는 날짜가 표시 범위 내인지

2. 새로고침 버튼 클릭

3. 브라우저 개발자 도구 → Network 탭에서 API 응답 확인

📙 물리치료사를 위한

실전 구축 & 대시보드 (3권)

    치료사 실적 대시보드

    공정한 평가를 위한 데이터

    힐링손 물리치료센터에는 김지훈 원장과 3명의 치료사가 있습니다. 월말이 되면 각 치료사의 실적을 정산하고 인센티브를 계산해야 합니다. 예전에는 엑셀로 하루 종일 걸리던 일이었죠.

    이제 데이터베이스가 있으니, 실시간으로 각 치료사의 실적을 확인하고 공정하게 평가할 수 있습니다. 치료사들도 자신의 실적을 직접 볼 수 있어서 동기부여가 됩니다.

    이번 장에서 만들 치료사 실적 대시보드에는 다음 내용이 포함됩니다:

    • 치료사별 치료 건수와 매출
    • 월별 실적 추이
    • 치료 항목별 분석
    • 환자 만족도 지표 (재방문율)
    • 인센티브 계산

    1. 대시보드 생성

    1. Metabase 상단 ➕ 새로 만들기 → 대시보드
    2. 이름: "👨‍⚕️ 치료사 실적 대시보드"
    3. 컬렉션: "힐링손 물리치료센터"
    4. 만들기 클릭

    2. 전체 실적 요약

    카드 1: 이번 달 전체 치료 건수

    SELECT COUNT(*) as 총치료건수
FROM treatment_sessions
WHERE DATE_FORMAT(session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND session_status = '완료';

    시각화: 숫자, 저장: "이번 달 총 치료 건수"

    카드 2: 이번 달 전체 매출

    SELECT COALESCE(SUM(b.total_amount), 0) as 총매출
FROM billings b
JOIN treatment_sessions ts ON b.session_id = ts.session_id
WHERE DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND b.payment_status = '완료';

    시각화: 숫자 (₩ 접두사), 저장: "이번 달 총 매출"

    카드 3: 치료사 수

    SELECT COUNT(*) as 치료사수
FROM therapists
WHERE is_active = 'Y';

    시각화: 숫자, 저장: "활성 치료사 수"

    카드 4: 치료사당 평균 매출

    SELECT ROUND(
(SELECT SUM(b.total_amount)
 FROM billings b
 JOIN treatment_sessions ts ON b.session_id = ts.session_id
 WHERE DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
   AND b.payment_status = '완료')
/
(SELECT COUNT(*) FROM therapists WHERE is_active = 'Y')
) as 평균매출;

    시각화: 숫자 (₩ 접두사), 저장: "치료사당 평균 매출"

    3. 치료사별 실적 비교

    카드 5: 치료사별 이번 달 실적

    SELECT
t.name as 치료사,
COUNT(ts.session_id) as 치료건수,
COALESCE(SUM(b.total_amount), 0) as 매출,
ROUND(COALESCE(SUM(b.total_amount), 0) / NULLIF(COUNT(ts.session_id), 0)) as 건당매출
FROM therapists t
LEFT JOIN treatment_sessions ts ON t.therapist_id = ts.therapist_id
AND DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
LEFT JOIN billings b ON ts.session_id = b.session_id
AND b.payment_status = '완료'
WHERE t.is_active = 'Y'
GROUP BY t.therapist_id, t.name
ORDER BY 매출 DESC;

    시각화: 테이블, 저장: "치료사별 이번 달 실적"

    카드 6: 치료사별 매출 비교 (막대)

    SELECT
t.name as 치료사,
COALESCE(SUM(b.total_amount), 0) as 매출
FROM therapists t
LEFT JOIN treatment_sessions ts ON t.therapist_id = ts.therapist_id
AND DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
LEFT JOIN billings b ON ts.session_id = b.session_id
AND b.payment_status = '완료'
WHERE t.is_active = 'Y'
GROUP BY t.therapist_id, t.name
ORDER BY 매출 DESC;

    시각화: 막대형, 저장: "치료사별 매출 비교"

    카드 7: 치료사별 치료 건수 비교

    SELECT
t.name as 치료사,
COUNT(ts.session_id) as 치료건수
FROM therapists t
LEFT JOIN treatment_sessions ts ON t.therapist_id = ts.therapist_id
AND DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
WHERE t.is_active = 'Y'
GROUP BY t.therapist_id, t.name
ORDER BY 치료건수 DESC;

    시각화: 막대형, 저장: "치료사별 치료 건수"

    4. 월별 추이 분석

    카드 8: 치료사별 월별 매출 추이

    SELECT
DATE_FORMAT(ts.session_date, '%Y-%m') as 월,
t.name as 치료사,
SUM(b.total_amount) as 매출
FROM treatment_sessions ts
JOIN therapists t ON ts.therapist_id = t.therapist_id
JOIN billings b ON ts.session_id = b.session_id
WHERE ts.session_status = '완료'
AND b.payment_status = '완료'
AND ts.session_date >= DATE_SUB(CURDATE(), INTERVAL 6 MONTH)
GROUP BY DATE_FORMAT(ts.session_date, '%Y-%m'), t.therapist_id, t.name
ORDER BY 월, t.name;

    시각화: 선형, X축: 월, 색상: 치료사, Y축: 매출, 저장: "치료사별 월별 매출"

    카드 9: 치료사별 월별 치료 건수 추이

    SELECT
DATE_FORMAT(ts.session_date, '%Y-%m') as 월,
t.name as 치료사,
COUNT(*) as 건수
FROM treatment_sessions ts
JOIN therapists t ON ts.therapist_id = t.therapist_id
WHERE ts.session_status = '완료'
AND ts.session_date >= DATE_SUB(CURDATE(), INTERVAL 6 MONTH)
GROUP BY DATE_FORMAT(ts.session_date, '%Y-%m'), t.therapist_id, t.name
ORDER BY 월, t.name;

    시각화: 선형, 저장: "치료사별 월별 건수"

    5. 치료 항목별 분석

    카드 10: 치료사별 항목 비중

    SELECT
t.name as 치료사,
f.fee_name as 항목,
COUNT(*) as 건수,
SUM(b.total_amount) as 매출
FROM treatment_sessions ts
JOIN therapists t ON ts.therapist_id = t.therapist_id
JOIN billings b ON ts.session_id = b.session_id
JOIN fee_items f ON b.fee_code = f.fee_code
WHERE DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
AND b.total_amount > 0
GROUP BY t.therapist_id, t.name, f.fee_code, f.fee_name
ORDER BY t.name, 매출 DESC;

    시각화: 테이블, 저장: "치료사별 항목 분석"

    카드 11: 치료사별 주력 항목 (가장 많이 한 치료)

    SELECT
t.name as 치료사,
f.fee_name as 주력항목,
COUNT(*) as 건수
FROM treatment_sessions ts
JOIN therapists t ON ts.therapist_id = t.therapist_id
JOIN billings b ON ts.session_id = b.session_id
JOIN fee_items f ON b.fee_code = f.fee_code
WHERE DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
AND b.total_amount > 0
GROUP BY t.therapist_id, t.name, f.fee_code, f.fee_name
HAVING COUNT(*) = (
SELECT MAX(cnt) FROM (
    SELECT t2.therapist_id, COUNT(*) as cnt
    FROM treatment_sessions ts2
    JOIN billings b2 ON ts2.session_id = b2.session_id
    WHERE ts2.therapist_id = t.therapist_id
      AND DATE_FORMAT(ts2.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
      AND ts2.session_status = '완료'
      AND b2.total_amount > 0
    GROUP BY ts2.therapist_id, b2.fee_code
) sub WHERE sub.therapist_id = t.therapist_id
)
ORDER BY t.name;

    시각화: 테이블, 저장: "치료사별 주력 항목"

    6. 환자 관리 지표

    카드 12: 치료사별 담당 환자 수

    SELECT
t.name as 치료사,
COUNT(DISTINCT ts.patient_id) as 담당환자수
FROM treatment_sessions ts
JOIN therapists t ON ts.therapist_id = t.therapist_id
WHERE DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
GROUP BY t.therapist_id, t.name
ORDER BY 담당환자수 DESC;

    시각화: 막대형, 저장: "치료사별 담당 환자 수"

    카드 13: 치료사별 신규 환자 유치

    SELECT
t.name as 치료사,
COUNT(DISTINCT ts.patient_id) as 신규환자수
FROM treatment_sessions ts
JOIN therapists t ON ts.therapist_id = t.therapist_id
WHERE DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
AND NOT EXISTS (
  SELECT 1 FROM treatment_sessions ts2
  WHERE ts2.patient_id = ts.patient_id
    AND ts2.session_date < DATE_FORMAT(CURDATE(), '%Y-%m-01')
)
GROUP BY t.therapist_id, t.name
ORDER BY 신규환자수 DESC;

    시각화: 막대형, 저장: "치료사별 신규 환자"

    카드 14: 치료사별 환자당 평균 방문 횟수

    SELECT
t.name as 치료사,
COUNT(*) as 총치료건수,
COUNT(DISTINCT ts.patient_id) as 환자수,
ROUND(COUNT(*) / COUNT(DISTINCT ts.patient_id), 1) as 환자당방문횟수
FROM treatment_sessions ts
JOIN therapists t ON ts.therapist_id = t.therapist_id
WHERE DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
GROUP BY t.therapist_id, t.name
ORDER BY 환자당방문횟수 DESC;

    시각화: 테이블, 저장: "환자당 평균 방문 횟수"

    💡 환자당 방문 횟수의 의미
    이 숫자가 높으면 환자가 꾸준히 다닌다는 뜻입니다. 치료 효과가 좋거나, 환자와의 신뢰 관계가 좋다는 지표가 될 수 있어요. 너무 낮으면 환자 이탈이 많다는 신호일 수 있습니다.

    7. 시간대별 분석

    카드 15: 치료사별 시간대 분포

    SELECT
t.name as 치료사,
CASE
    WHEN HOUR(ts.start_time) < 12 THEN '오전 (9-12)'
    WHEN HOUR(ts.start_time) < 15 THEN '오후1 (12-15)'
    WHEN HOUR(ts.start_time) < 18 THEN '오후2 (15-18)'
    ELSE '저녁 (18-20)'
END as 시간대,
COUNT(*) as 건수
FROM treatment_sessions ts
JOIN therapists t ON ts.therapist_id = t.therapist_id
WHERE DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
GROUP BY t.therapist_id, t.name, 시간대
ORDER BY t.name;

    시각화: 누적 막대형, 저장: "치료사별 시간대 분포"

    카드 16: 치료사별 요일 분포

    SELECT
t.name as 치료사,
CASE DAYOFWEEK(ts.session_date)
    WHEN 1 THEN '일'
    WHEN 2 THEN '월'
    WHEN 3 THEN '화'
    WHEN 4 THEN '수'
    WHEN 5 THEN '목'
    WHEN 6 THEN '금'
    WHEN 7 THEN '토'
END as 요일,
COUNT(*) as 건수
FROM treatment_sessions ts
JOIN therapists t ON ts.therapist_id = t.therapist_id
WHERE DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
GROUP BY t.therapist_id, t.name, DAYOFWEEK(ts.session_date)
ORDER BY t.name, DAYOFWEEK(ts.session_date);

    시각화: 히트맵 또는 테이블, 저장: "치료사별 요일 분포"

    8. 인센티브 계산

    힐링손의 인센티브 정책을 가정해봅시다:

    기준인센티브
    월 매출 200만원 이상초과분의 10%
    신규 환자 유치1인당 5만원
    월 치료 건수 50건 이상초과 건당 1만원

    카드 17: 인센티브 계산표

    SELECT
t.name as 치료사,
COALESCE(SUM(b.total_amount), 0) as 매출,
COUNT(DISTINCT ts.session_id) as 치료건수,
(SELECT COUNT(DISTINCT ts2.patient_id)
 FROM treatment_sessions ts2
 WHERE ts2.therapist_id = t.therapist_id
   AND DATE_FORMAT(ts2.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
   AND ts2.session_status = '완료'
   AND NOT EXISTS (
       SELECT 1 FROM treatment_sessions ts3
       WHERE ts3.patient_id = ts2.patient_id
         AND ts3.session_date < DATE_FORMAT(CURDATE(), '%Y-%m-01')
   )) as 신규환자,
-- 매출 인센티브: 200만원 초과분의 10%
GREATEST(0, (COALESCE(SUM(b.total_amount), 0) - 2000000)) * 0.1 as 매출인센티브,
-- 신규환자 인센티브: 1인당 5만원
(SELECT COUNT(DISTINCT ts2.patient_id) * 50000
 FROM treatment_sessions ts2
 WHERE ts2.therapist_id = t.therapist_id
   AND DATE_FORMAT(ts2.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
   AND ts2.session_status = '완료'
   AND NOT EXISTS (
       SELECT 1 FROM treatment_sessions ts3
       WHERE ts3.patient_id = ts2.patient_id
         AND ts3.session_date < DATE_FORMAT(CURDATE(), '%Y-%m-01')
   )) as 신규인센티브,
-- 건수 인센티브: 50건 초과분 × 1만원
GREATEST(0, COUNT(DISTINCT ts.session_id) - 50) * 10000 as 건수인센티브
FROM therapists t
LEFT JOIN treatment_sessions ts ON t.therapist_id = ts.therapist_id
AND DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
LEFT JOIN billings b ON ts.session_id = b.session_id
AND b.payment_status = '완료'
WHERE t.is_active = 'Y'
GROUP BY t.therapist_id, t.name
ORDER BY 매출 DESC;

    시각화: 테이블, 저장: "인센티브 계산표"

    카드 18: 총 인센티브

    SELECT
t.name as 치료사,
(
    GREATEST(0, (COALESCE(SUM(b.total_amount), 0) - 2000000)) * 0.1 +
    (SELECT COUNT(DISTINCT ts2.patient_id) * 50000
     FROM treatment_sessions ts2
     WHERE ts2.therapist_id = t.therapist_id
       AND DATE_FORMAT(ts2.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
       AND ts2.session_status = '완료'
       AND NOT EXISTS (
           SELECT 1 FROM treatment_sessions ts3
           WHERE ts3.patient_id = ts2.patient_id
             AND ts3.session_date < DATE_FORMAT(CURDATE(), '%Y-%m-01')
       )) +
    GREATEST(0, COUNT(DISTINCT ts.session_id) - 50) * 10000
) as 총인센티브
FROM therapists t
LEFT JOIN treatment_sessions ts ON t.therapist_id = ts.therapist_id
AND DATE_FORMAT(ts.session_date, '%Y-%m') = DATE_FORMAT(CURDATE(), '%Y-%m')
AND ts.session_status = '완료'
LEFT JOIN billings b ON ts.session_id = b.session_id
AND b.payment_status = '완료'
WHERE t.is_active = 'Y'
GROUP BY t.therapist_id, t.name
ORDER BY 총인센티브 DESC;

    시각화: 막대형, 저장: "치료사별 총 인센티브"

    9. 개인별 상세 카드

    카드 19: 치료사 프로필 (변수 사용)

    특정 치료사를 선택해서 상세 정보를 볼 수 있는 카드입니다.

    SELECT
t.name as 치료사명,
t.specialization as 전문분야,
t.hire_date as 입사일,
TIMESTAMPDIFF(MONTH, t.hire_date, CURDATE()) as 근속개월,
(SELECT COUNT(*) FROM treatment_sessions ts
 WHERE ts.therapist_id = t.therapist_id AND ts.session_status = '완료') as 누적치료건수,
(SELECT COUNT(DISTINCT patient_id) FROM treatment_sessions ts
 WHERE ts.therapist_id = t.therapist_id AND ts.session_status = '완료') as 누적담당환자
FROM therapists t
WHERE t.name = {{치료사명}};

    변수 {{치료사명}}을 추가하면 드롭다운으로 치료사를 선택할 수 있습니다.

    11. 필터 설정

    월 선택 필터

    1. 편집 모드 → 필터 추가
    2. 시간 필터 (월 단위)
    3. 기본값: 이번 달
    4. 모든 카드에 연결

    치료사 선택 필터

    1. 필터 추가 → 카테고리
    2. 치료사 이름 컬럼 연결
    3. 모두 선택 / 개별 선택 가능

    이렇게 하면 특정 치료사만 골라서 볼 수도 있고, 전체를 비교할 수도 있습니다.

    12. 활용 시나리오

    월초: 목표 설정

    1. 지난달 실적 확인
    2. 치료사별 강점/약점 파악
    3. 이번 달 목표 설정

    월중: 진행 상황 체크

    1. 목표 대비 진척률 확인
    2. 뒤처지는 치료사 지원
    3. 잘하는 치료사 노하우 공유

    월말: 정산 및 평가

    1. 최종 실적 확인
    2. 인센티브 계산
    3. 개인별 피드백
    💡 공정한 평가를 위해
    • 데이터로 평가하면 감정적 분쟁이 줄어듭니다
    • 치료사들도 자신의 실적을 실시간으로 확인 가능
    • 투명한 인센티브 정책으로 동기부여

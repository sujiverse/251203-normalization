    힐링손 물리치료센터 이야기

    김지훈 원장의 고민

    김지훈 원장은 올해로 물리치료사 경력 10년차입니다. 대학병원에서 5년, 개인 클리닉에서 3년을 일한 뒤, 2년 전에 독립해서 자신의 클리닉을 열었습니다. 이름은 "힐링손 물리치료센터". 손으로 치료하는 도수치료가 전문이라 이런 이름을 붙였어요.

    처음에는 환자가 많지 않았습니다. 하루에 5명 오면 많이 오는 날이었죠. 그때는 엑셀 파일 하나로 충분했습니다. 환자 이름, 연락처, 치료 날짜, 받은 돈... 한 시트에 다 적어도 됐어요.

    그런데 2년이 지난 지금, 상황이 완전히 달라졌습니다.

    2년 전과 지금의 비교

    항목2년 전 (개원 초기)현재
    하루 환자 수3~5명25~35명
    누적 환자 수약 30명400명 이상
    직원 수원장 혼자원장 + 치료사 3명
    월 매출300~400만원2,500만원 이상
    데이터 관리엑셀 파일 1개엑셀 파일 6개 + 종이 차트

    성장한 건 좋은 일이죠. 하지만 데이터 관리가 점점 악몽이 되어가고 있습니다.

    김지훈 원장의 하루

    월요일 아침 8시 30분. 김지훈 원장은 클리닉에 출근해서 컴퓨터를 켭니다.

    "오늘 예약 환자가 누구누구지?"

    예약관리.xlsx 파일을 엽니다. 30초 걸립니다. 파일이 무거워졌거든요. 오늘 예약된 환자 목록을 확인합니다. 28명이네요.

    "김영희 환자분... 지난번에 뭘로 치료했더라?"

    치료기록.xlsx 파일을 엽니다. 또 30초. Ctrl+F로 "김영희"를 검색합니다. 같은 이름이 3명이 나옵니다. 생년월일을 확인해서 맞는 환자를 찾습니다. 지난 치료 기록을 확인합니다.

    "이 환자분 미수금 있었던 것 같은데..."

    수납관리.xlsx 파일을 엽니다. 또 30초. 다시 검색. 아, 3만원 미수금이 있네요.

    이제 겨우 한 환자 정보를 파악했습니다. 이걸 28번 반복해야 합니다.

    월말의 악몽

    매달 마지막 주가 되면 김지훈 원장은 한숨이 나옵니다. 월말 정산을 해야 하거든요.

    해야 할 일들:

    • 이번 달 총 매출 계산
    • 치료사별 치료 건수 집계
    • 치료사별 인센티브 계산
    • 미수금 현황 파악
    • 보험 청구 준비

    이걸 하려면 여러 엑셀 파일을 열어서 VLOOKUP 걸고, 피벗 테이블 만들고, 수식 오류 잡고... 보통 반나절에서 하루가 걸립니다. 그것도 야근해서요.

    "분명히 치료 기록에는 30건인데, 수납 기록에는 28건이야. 2건은 어디 갔지?"

    이런 불일치가 자주 발생합니다. 찾아보면 대부분 입력 실수예요. 한 파일에는 입력했는데 다른 파일에는 깜빡한 거죠.

    엑셀의 한계

    김지훈 원장이 겪고 있는 문제들, 혹시 공감되시나요? 정리해보면 이렇습니다:

    문제 1: 데이터가 여기저기 흩어져 있다

    현재 사용 중인 파일들:

    • 환자명단.xlsx - 환자 기본 정보
    • 예약관리.xlsx - 예약 현황
    • 치료기록.xlsx - 일자별 치료 내용
    • 수납관리.xlsx - 청구 및 수납
    • 치료사실적.xlsx - 치료사별 실적
    • 인센티브계산.xlsx - 월별 인센티브
    • + 종이 SOAP 차트 수백 장

    하나의 정보를 보려면 여러 파일을 열어서 대조해야 합니다. 시간도 오래 걸리고, 실수하기도 쉽습니다.

    문제 2: 같은 정보를 여러 번 입력한다

    새 환자가 오면:

    1. 환자명단.xlsx에 이름, 연락처, 생년월일 입력
    2. 예약관리.xlsx에 이름, 연락처 입력
    3. 치료기록.xlsx에 이름, 진단명 입력
    4. 수납관리.xlsx에 이름, 금액 입력

    같은 환자 이름을 4번이나 입력합니다. 연락처도 여러 번 입력하고요. 시간 낭비인 데다가, 오타가 나면 나중에 검색이 안 됩니다.

    문제 3: 데이터 불일치가 생긴다

    환자가 전화번호를 바꿨어요. 환자명단.xlsx는 수정했는데, 예약관리.xlsx는 깜빡했습니다. 나중에 연락하려고 보니 파일마다 전화번호가 다릅니다. 어느 게 맞는 거지?

    문제 4: 복잡한 질문에 답하기 어렵다

    이런 질문에 답해야 한다고 생각해보세요:

    "지난달에 3회 이상 치료받은 환자 중에서, 허리 디스크 진단을 받은 환자는 몇 명이고, 그들의 총 치료비는 얼마야?"

    엑셀로 이걸 구하려면... 여러 파일을 열고, VLOOKUP 걸고, 필터 걸고, 피벗 만들고... 엑셀 고수도 한참 걸립니다.

    문제 5: 파일이 커지면 느려진다

    2년치 데이터가 쌓이니 엑셀 파일이 10MB를 넘어갑니다. 파일 여는 데 30초, 저장하는 데 20초, 검색하는 데 10초... 하루에 이런 대기 시간만 합쳐도 30분이 넘습니다.

    문제 6: 동시 작업이 안 된다

    박민수 치료사가 치료기록.xlsx를 열고 있으면, 이서연 치료사는 그 파일을 수정할 수 없습니다. "다른 사용자가 사용 중입니다" 메시지가 뜨죠. 그래서 각자 자기 파일을 만들기 시작했고, 나중에 합치느라 또 시간이 듭니다.

    결심

    어느 날 밤, 또 월말 정산으로 야근을 하던 김지훈 원장이 결심합니다.

    "더 이상 엑셀로 버티지 말자. 제대로 된 데이터베이스를 만들자. 그리고 대시보드를 만들어서 클릭 한 번에 현황을 볼 수 있게 하자!"

    마침 물리치료사 커뮤니티에서 "물리치료사를 위한 데이터베이스" 시리즈를 발견했습니다. 1권과 2권을 읽고 개념을 익혔어요. 이제 3권을 보면서 직접 만들어볼 차례입니다.

    우리도 김지훈 원장과 함께 힐링손 물리치료센터의 데이터베이스를 만들어봅시다!

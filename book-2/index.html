    📁 pt_clinic 데이터베이스

[patients]         환자 정보
[therapists]       치료사 정보
[treatment_sessions]  치료 세션
[diagnosis_codes]  진단 코드 목록
[patient_diagnoses]  환자-진단 연결
[billings]         청구/수납
[fee_items]        청구 항목
[soap_notes]       SOAP 기록
[insurances]       보험사 정보

    쿼리 읽는 법

    SELECT 뭘_볼건지           -- 1. 어떤 컬럼
FROM 어느_테이블에서        -- 2. 어느 테이블
JOIN 다른_테이블           -- 3. 테이블 연결
WHERE 조건                 -- 4. 필터 조건
GROUP BY 그룹_기준         -- 5. 집계 기준
ORDER BY 정렬_기준         -- 6. 정렬
